# Restaurant Landing Page

## Description

Fully-functional restaurant landing page for Gericht that displays,

-   An About Us section.
-   Chef info.
-   Find Us section.
-   Photo Gallery from Instagram.
-   Restaurant Laurels.
-   Contact information / Restaurant hours.

### Showcases

-   HTML5 and CSS3
-   React

## Getting Started

```
$ npm install
$ npm start
```
